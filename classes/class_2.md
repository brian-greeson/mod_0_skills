## Class: MenuItem

### Attributes:
- name (string)
- onMenu (bool)
- price (float)
- ingredients (array)

### Methods:
- changeName (modifies the name attribute)
- addToMenu  (sets onMenu to true)
- removeFromMenu (sets onMenu to false)
- setPrice (modifies the price attribute)
- addIngredient (adds a string to the array of ingredients)
